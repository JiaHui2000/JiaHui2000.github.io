Node.js使用小记

发表于 2022-07-09 阅读次数： 阅读次数： Valine：
记录 Node.js 使用的一些问题。

NVM

使用 hexo douban 插件时发现 v15 版本不能导入电影，故采用此软件。

安装

  1. 下载 Download

  2. 选择 nvm 安装路径和 nodejs 安装路径。

使用

  1. nvm list available 查看可用版本。注意此查询只是部分版本，如果想访问完整版本，可以访问 https://nodejs.org/download/release
  2. nvm install 版本号 安装指定版本。

问题

exit status 1

exit status 1:乱码，或出现exit status 1: You do not have sufficient privilege to perform this operation.这里需要用管理员身份打开cmd命令行工具，再执行nvm use ***即可。
